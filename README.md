Имеются четыре файлика, которые прошли через рефакторинг. 

func_main.kt отвечает за вывод и запуск самой программы(главная функция)

Movable.kt - это интерфейс, от которого берут другие файлы поля, типо шаблона чтоли

Human.kt отвечает за классы, конструкторы и методы, в нём есть класс Человек, которые наследует поля у интерфейса Movable.kt

Driver.kt есть отдельный класс, отвечающий за Водителя, в нём написано только процесс работы с Водителем, его поток, координаты перемещения и скорость

Сама программа выводит Людей, и в потоках с паузами в 1 секунду, работает движение этих Людей, а также в конце включен водитель, которые едет также по координатам и со своей скоростью
