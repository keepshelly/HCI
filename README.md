#Крахмальный Кирилл Вячеславович ИКС-433

 func_main.kt
 * Главная функция программы (точка входа).
 * Создает массив объектов класса Human и один объект класса Driver.
 * Запускает метод move() для каждого человека и водителя.
 * После завершения движения программа ожидает 2 секунды перед завершением.
 * Используемые классы:
 * Human
 * Driver


Human.kt
 * Базовый класс "Человек".
  Содержит основные свойства: имя, фамилия, отчество, номер группы, координаты (x, y).
  
 * Конструктор:
  - Инициализирует свойства и выводит сообщение о создании объекта.
  
  Методы:
  - move(): запускает движение в отдельном потоке со случайными координатами.
  - moveTo(_toX, _toY): перемещает человека в указанные координаты.
  - randomMove(): перемещает человека в случайные координаты от 1 до 100.
  - gaussmarkov(): реализует движение по модели Гаусса-Маркова с учетом скорости и направления.
  
 * Используемые библиотеки:
  - kotlin.math.*
  - kotlin.random.Random

Movable.kt
 * Интерфейс "Movable" (Перемещаемый).
  Определяет контракт для объектов, способных двигаться.
  
 * Свойства:
  - X: текущая координата X (Double)
  - Y: текущая координата Y (Double)
  - Speed: скорость перемещения (Double)
  
 * Метод:
  - move(): должен быть реализован в классе, поддерживающем интерфейс.

 Driver.kt
 * Класс "Водитель", наследуется от Human.
 * Дополнительное свойство:
  - Prava: категория прав (например, "C")
  
 * Переопределяет метод move():
  - Увеличивает координаты x на 15, y на 10.
  - Выводит сообщение о движении водителя.
  - Запускается в отдельном потоке.
  
 * Конструктор:
  - Принимает имя, фамилию, отчество и категорию прав.
  - Номер группы устанавливается в -1 по умолчанию.
