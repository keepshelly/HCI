**Модель движения со стохастической инерцией**

*Общая концепция:*
Поведение объекта определяется балансом между сохранением предыдущего состояния и случайными воздействиями. Модель использует параметр управления α в диапазоне от 0 до 1, где крайние значения соответствуют разным режимам движения.

*Параметры управления:*
- α = 1: движение полностью инерционное (сохраняется предыдущая траектория)
- α = 0: движение полностью случайное (отсутствует память о предыдущем состоянии)
- 0 < α < 1: смешанный режим с элементами инерции и случайности

*Математический аппарат:*

**Обновление скорости:**
vₜ₊₁ = α·vₜ + (1-α)·v̄ + √(1-α²)·wᵥ

**Обновление направления:**
θₜ₊₁ = α·θₜ + (1-α)·θ̄ + √(1-α²)·wθ

**Расчет координат:**
xₜ₊₁ = xₜ + vₜ₊₁·cos(θₜ₊₁)
yₜ₊₁ = yₜ + vₜ₊₁·sin(θₜ₊₁)

*Обозначения параметров:*
- v̄, θ̄ — базовые (средние) значения скорости и направления
- wᵥ, wθ — случайные компоненты (гауссовский шум)
- α — коэффициент инерционности движения
- t, t+1 — текущий и следующий моменты времени

*Особенности модели:*
Модель позволяет гибко настраивать степень случайности движения через единственный параметр α, что делает ее удобной для моделирования различных сценариев поведения объектов.
